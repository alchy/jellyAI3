JellyAI — architektura README webu

README dokumentačního webu

Co to je

Propojený HTML web s kompletní architektonickou dokumentací projektu jellyAI3: index.html + 17 kapitol (samostatné stránky, interní odkazy, obsah na každé stránce). Popisuje skutečný stav kódu ke commitu a63e83f (větev otazkovy-graf, 2026-07-20).

Jak číst offline

Stačí otevřít index.html v prohlížeči — web nemá žádné externí závislosti (CSS i skripty jsou lokální). Funguje i přes file://.

Mermaid diagramy

Diagramy jsou v HTML vložené jako <pre class="mermaid"> — tj. čitelný textový zápis, který se po načtení stránky vykreslí lokálním souborem mermaid.min.js (v tomto adresáři; verze 10.9.1, staženo z npm balíčku mermaid). Pokud by soubor chyběl nebo se nenačetl, diagramy zůstanou zobrazené jako čitelný mermaid zdroj — o obsah nepřijdete.

Doplnění/aktualizace souboru jedním řádkem:

curl -sL -o docs/architektura-web/mermaid.min.js \
  https://cdn.jsdelivr.net/npm/mermaid@10.9.1/dist/mermaid.min.js

Odkazy na zdrojáky

Odkazy tvaru jellyai/iris/qgraph.py:55 vedou relativně do repozitáře (../../jellyai/…). Čísla řádků odpovídají commitu a63e83f — po dalším vývoji se mohou posunout; soubor a jméno funkce zůstávají spolehlivým vodítkem.

Doprovodné soubory

  • postrehy-refaktor.md — review postřehů k členění projektu (DRY / konzistence / pochopitelnost / členění), sepsané při tvorbě této dokumentace. Nejde o součást učebnice — je to podklad pro budoucí refaktor.

Aktualizace dokumentace

Web je statický — při větších změnách architektury aktualizujte dotčené kapitoly ručně a srovnejte s aktuálními zdroji pravdy (docs/ARCHITEKTURA.md, HANDOVER.md, BACKLOG.md, specy). Metriky v kapitolách 1 a 15 nesou konkrétní čísla benchmarků — po změnách je přeměřte.
